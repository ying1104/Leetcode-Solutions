#### 397. Integer Replacement

```java
class Solution {
    public int integerReplacement(int n) {
        int step = 0;
        while (n != 1) {
            if (n == Integer.MAX_VALUE) {
                n = n / 2 + 1;
                step += 2;
                continue;
            }
            if (n % 2 == 0) n = n / 2;
            else if ((n - 1) % 4 == 0 || n == 3)
                n--;
            else n++;
            step++;
            
        }
        return step;
    }

}
```
