#### 371. Sum of Two Integers

```java
// Solution 1: Iterative
class Solution {
    public int getSum(int a, int b) {
        while (b != 0) {
            int temp = a;
            a = a ^ b;
            b = (b & temp) << 1;
        }
        return a;
    }
}
```

```java
// Solution 2: Recursion
class Solution {
    public int getSum(int a, int b) {
        return b == 0 ? a : getSum(a ^ b, (a & b) << 1);
    }
}
```
