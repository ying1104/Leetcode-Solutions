#### 122. Best Time to Buy and Sell Stock II

```java
class Solution {
    public int maxProfit(int[] prices) {
        int res = 0;
        for (int i = 1; i < prices.length; i++) {
            if (prices[i] < prices[i - 1]) continue;
            res += prices[i] - prices[i - 1];
        }
        return res;
    }
}
```
