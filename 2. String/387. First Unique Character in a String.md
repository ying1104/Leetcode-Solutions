#### 387. First Unique Character in a String



```java
class Solution {
    public int firstUniqChar(String s) {
        if (s == null || s.length() == 0) return -1;
        int freq[] = new int[26];
        
        int L = s.length();
        for (int i = 0; i < L; i++) {
            int toAdd = (int) s.charAt(i) - (int) 'a';
            freq[toAdd]++;
        }
        for (int i = 0; i < L; i++) {
            if (freq[(int) s.charAt(i) - (int) 'a'] == 1) {
                return i;
            }
        }
        return -1;
       
    }
}
```
